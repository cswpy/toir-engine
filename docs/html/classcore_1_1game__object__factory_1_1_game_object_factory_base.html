Detailed Description

GameObjectFactoryBase is an abstract class that provides a common interface and utility functions for creating game objects such as backgrounds, images, and buttons. Subclasses are expected to implement the abstract methods to provide actual GameObject construction logic.

Usage typically involves:

  1. Calling init(renderer) once at startup to set the SDL_Renderer.
  2. Using createBackground, createImage, and createButton (or createCenteredButton) to generate various UI elements and objects.

Create a button GameObject with the specified text, rectangle, and click action. The button will typically change colors when hovered over by the mouse, and execute the provided onClick delegate when clicked.

Params: text = The text to display on the button. rect = An SDL_Rect specifying the button's position and size. onClick = A delegate (function pointer) that is called when the button is clicked. buttonColor = The default color of the button (when not hovered). hoverColor = The color of the button when hovered.

Returns: A GameObject representing the interactive button.

Create a button that is automatically centered horizontally in the window. The button's vertical position (Y coordinate) is specified, while its width and height can be provided or fallback to default values.

Params: text = The button text. y = The y-coordinate at which to place the button. width = The button's width (default: buttonConfig.defaultWidth). height = The button's height (default: buttonConfig.defaultHeight). onClick = A delegate called when the button is clicked (optional). buttonColor = The default button color (optional). hoverColor = The button hover color (optional).

Returns: A GameObject representing the centered button.
