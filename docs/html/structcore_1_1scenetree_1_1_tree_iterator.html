Detailed Description

TreeIterator is a forward iterator that allows traversal of Node instances in a scene tree, focusing on those that have associated GameObject instances.

Constructor & Destructor Documentation

◆ this()

core::scenetree::TreeIterator::this ( Node root)
inline

Construct a TreeIterator from a given root node. This will retrieve all nodes in order, filter out those without a GameObject, and store them for iteration.

Params: root = The node from which to start the traversal.
