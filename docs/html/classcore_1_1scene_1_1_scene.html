Detailed Description

The Scene class is an abstract base class that defines the general interface and responsibilities of a scene within the game. Scenes contain a SceneTree that organizes GameObject instances and other elements into a structured hierarchy.

Every Scene:

  • Knows how to render its contents to the screen.
  • Can process input events, allowing user interaction.
  • May implement optional update and cleanup logic.

Subclasses should provide concrete implementations of render() and input() methods, and may override update() and cleanup() as needed.
