Detailed Description

IGameScene is an abstract extension of the Scene class, intended to represent active gameplay scenarios. It requires concrete implementations of the render(), input(), update(), and cleanup() methods, ensuring that any subclass provides the full set of functionality needed to run a game scene.

By inheriting from Scene, IGameScene also benefits from a SceneTree for organizing its objects, as well as a reference to the main SDL_Renderer for drawing graphical content.
