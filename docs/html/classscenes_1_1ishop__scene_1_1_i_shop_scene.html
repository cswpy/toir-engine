Detailed Description

IShopScene is an abstract class that represents a shop interface within the game. It requires specific shop-related methods to be implemented, such as setting up available items and retrieving the player's points. By handling rendering, input, updating, and cleanup, IShopScene ensures a smooth and interactive shopping experience for the player.
