Detailed Description

IResourceManager is an interface that defines the methods required for loading, retrieving, and cleaning up various game resources.

Key responsibilities:

  • Providing a singleton instance for global resource management access.
  • Loading and returning textures, fonts, sounds, and music.
  • Ensuring proper cleanup of resources when they are no longer needed.

By implementing this interface, developers can create a centralized and efficient resource management system suitable for their game's needs.

◆ cleanup()

void core::resource_manager::IResourceManager::cleanup ( )

Clean up all loaded resources managed by this resource manager. This typically involves freeing textures, fonts, sounds, and music that have been cached. It should be called when shutting down the game or when the resources are no longer needed.

Retrieve a TTF_Font at a given size. The font may be loaded and cached to avoid redundant file reads. Multiple sizes of the same font may be cached separately.

Params: path = The file path to the font resource (e.g., a TTF file). size = The font size (in points) to load.

Returns: A pointer to the loaded TTF_Font. On failure, returns null.
