Detailed Description

IMenuScene is an abstract extension of the Scene class, designed specifically for menu screens. Implementations must provide UI setup, rendering, input handling, updates, and cleanup logic. This ensures that each menu scene can present a cohesive, navigable interface to players and integrate smoothly into the game's flow.
