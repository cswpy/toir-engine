Detailed Description

The IDiceManager abstract class manages a collection of dice GameObjects along with their values, visual states, and user interactions. Implementations of this class must provide the logic for:

  • Setting up dice positions within a scene
  • Initiating and handling dice rolls
  • Updating dice states over time (e.g., for animations)
  • Responding to mouse events to drag and drop dice

By subclassing IDiceManager, developers can create custom dice mechanics, UI, and behaviors tailored to their game's requirements.
