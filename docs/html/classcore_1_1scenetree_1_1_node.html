Detailed Description

The Node class represents a single node in the scene tree. Each node can have zero or more children, a parent, and a GameObject associated with it.

Nodes facilitate hierarchical scene management:

  • They allow adding/removing children.
  • They support iteration and traversal through the TreeIterator.
  • They enable calling methods on all contained GameObject instances.

void core::scenetree::Node::callOnElements ( string funcName,

inline

Call a specified member function, if present, on every GameObject within this node's subtree.

This uses compile-time reflection to check if the GameObject type has a method with the given funcName. If it does, the method is called with the provided arguments args.

Params: funcName = The name of the method to call on each GameObject. args = The arguments to pass to that method.
